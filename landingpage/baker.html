Baker - One Page Bootstrap Template


Baker is a creative, multipurpose free bootstrap html5 template designed for professionals and agencies. This free html5 template is suitable for any business, corporate, portfolio, blog and any kind of website. It is clean and has a professional design and perfect solution for web designers.

The template is powered by Bootstrap framework. This free bootstrap html5 template can be easily customized and mobile friendly. This template have lot of amazing features like Flexible, well crafted, beautiful and unique concepts, well documented code, a bunch of components.

Features:

  • Built with latest Bootstrap version
  • Clean & Modern Design
  • NodeJS/NPM and Gulp version available (pro unlimited and membership users only)
  • Easy to customize, clean and commented code
  • PHP/Ajax contact form script (pro version only)
  • Portfolio with lightbox support
  • Portfolio details lightbox
  • Fully customizable
  • One pager design
  • Smooth navigation and links scroll
  • Cross Browser compatible
  • Beautiful mobile menu
  • Responsive Layout
